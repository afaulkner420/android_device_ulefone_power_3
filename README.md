Device repository for Ulefone Power 3 (LineageOS 14.1)
===========================

Getting Started
---------------

Initialize a repository with LineageOS:

    repo init -u git://github.com/LineageOS/android.git -b cm-14.1
    repo sync


Build the code:

    source build/envsetup.sh
    time brunch power_3
